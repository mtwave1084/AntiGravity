🔁 起動時のジョブ登録フロー

start.bat でセッション起動
↓ SessionStart フック
ccc-boot が実行される
↓ ステップ4
ccc-jobs スキルを呼び出す
↓
JOBS.yaml を読み込み、active: true のジョブを CronCreate に登録
↓ 毎朝7:30
ai-news スキルが自動実行 → HTMLをDiscordに添付送信
⚠ セッション限定の注意点

CronCreate のジョブはこのセッション内のみ有効（ディスクに保存されない）。Claude Code を終了すると消える。
→ start.bat → ccc-boot → ccc-jobs の流れで毎回再登録される設計になってるよ。
また、自動期限は登録から7日間。7日後に自動削除されるため、再起動して再登録が必要。

✓ 新しいジョブの追加方法

JOBS.yaml に新しいエントリを追加して active: true にするだけ。
次回セッション起動時（start.bat）に ccc-boot → ccc-jobs が自動登録してくれる。
