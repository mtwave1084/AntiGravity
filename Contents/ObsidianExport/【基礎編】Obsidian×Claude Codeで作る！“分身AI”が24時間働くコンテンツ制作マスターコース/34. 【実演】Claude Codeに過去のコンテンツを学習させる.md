そうすると、今、メールマガジンの中に、この過去作ったメールマガジンのMARKDOWNファイルが格納されていきました。

このような形で今一つしか入っていませんが、これが2つ、5つ、10個、100個、1000個という風にどんどんどんどんと蓄積していただける

と。

このメルマガデータというのをクロードコードに読み込んでもらって、皆さんのライティングの癖や伝える流れとか、どんな口調でライティングしている

のかといった部分をマニュアル化することができます。

書いたようなコンテンツがどんどん作成できていくという形になっていきます。
