この違いを理解するうえで、まず覚えておいてほしい言葉が2つあります。

それがフロー型と蓄積型です。

順番に説明をしていきます。

まず、フロー型とは何でしょうか。

チャット。

GPTやクロード、GEMINIなどのAIは、基本的にこのフロー型のAIになります。

どういうことかというと、例えば今日チャットGPTにこのブログの記事を参考にこんな内容で記事を書いてとお願いしたとします。

その場ではちゃんと出力をしてくれるかもしれません。

でも、次の日、翌月といった形で新しいテーマを書こうと思ったら、またチャット。

GPTに1から前の記事を読み込ませて構成を伝えたり、毎回全部手直しやり直しをすることが多いと思います。

つまり、どれだけAIに教えても1定数しか記憶されないといったのが今の現状のAIですよね。

例えるなら、ザルに水を注ぐようなもので、知識がどんどんと流れ落ちていってしまう。

これがフロー型の現在のAIの限界になります。

一方で、オブシディアンとクロード構造の仕組みはどうでしょうか。

こちらは完全に蓄積型になります。

例えば、あなたが与えた情報で日々のメモや気づき、書いた台本のデータ、今までのSNSの投稿文、構想中のアイディア。

こういった全ての情報をAIがそのまま記憶をして学習をしてくれます。

そして、蓄積された情報を元に、今度はこんな記事を書きたいんだけれども、次の台本はこんなテーマでと相談をすると、過去のやり取り

やあなたの価値観も踏まえた上で、的確なアウトプットを返してくれるようになるんです。

つまり、あなたの記憶をAIが代わりに持っていってくれるようなイメージです。

しかも、この蓄積というのは人間と違って忘れることがありません。

例えば、前に書いたメモどこ行ったっけとか、このリサーチどこに保存したっけといった、そんなストレスが一切なくなるということです。

あなたの学びや気づき、コンテンツ制作の癖まで全部まとめて一つの場所に記憶をしてくれる。

それがオブシディアンの役割になってきます。

さらにもう一つ重要な違いがあります。

それが情報をコンテンツ化するまでの流れです。

従来のAIは、例えばアイデアや文章のたたき台を出してくれることはあっても。

最終的にそれをどう活用してコンテンツに落とし込んでという作業は、結局僕たち人間がやらないといけなかったんですよね。

でも、この講座で使うクローズドコードというツールは違います。

クローズドコードは、オブシディアンに蓄積された情報を元に、実際のコンテンツを自動で組み立ててくれるんです。

分かりやすく例えると、オブシディアンは冷蔵庫。

いろんな情報やコンテンツ制作に必要な材料を蓄える場所。

蔵人コードはシェフ材料を取り出して自動で調理をして一つのコンテンツにしてくれる存在。

つまり、どちらか一方だけを使うのは不完全で、この2つを連携させて初めて自分の分身AIが完成するというわけです。

とはいえ、AIに情報を学習させるのは手間じゃないと思うかもしれません。

確かに最初はちょっと大変なんですよね。

でも、その手間をかけずに従来のやり方を続けていても、あなたにしか作れないオリジナル性のあるコンテンツは。

AIからは絶対に生まれません。

AIが進化して誰でもコンテンツが作れる時代だからこそ、求められるのはあなたの1次情報だからです。

あなたの体験談や思想、理念、失敗談、そういったレアなナレッジをAIに学習させることで、はじめて本物のコンテンツが作れるようになります。

ここで言うナレッジというのは、ただの知識や情報ではありません。

あなたが実際に経験をして得た気づきや洞察の事です。

このナレッジをいかにAIに蓄積させられるかが、これからの時代におけるクリエイターの勝負どころになります。

まとめると、従来のAIというのは、その場限りのやり取りしかできずに情報も積み上がらないし、コンテンツ化も自分でやる必要がありまし

た。

一方で、これから解説をするオブシディアンとクロードコードの組み合わせは、あなたの価値観や知識、構成方法を丸ごと学習して。

蓄積解釈コンテンツ化までを一気通貫で自動化してくれる。

そして、この連携は、現時点では他のどのAIツールでも実現ができない仕組みになります。
