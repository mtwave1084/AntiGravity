# ccc-handoff

現在のセッション状態をディスクに保存し、次回セッションへ引き継ぐスキル。
セッション終了前や長い作業の区切りに呼び出す。

## トリガー
- セッションを終了する前（ユーザーが「終わり」「おやすみ」「また後で」などと言ったとき）
- 長い作業を完了したとき
- CLAUDE.md の「セッション終了時のルール」に従って自動実行

## やること

### 1. 現在の状態を把握する
TaskList を呼んで現在のタスク一覧（in_progress / pending）を確認する。

### 2. handoff.md を書く
`C:\Users\mt_wa\projects\agy\.claude\handoff.md` に以下の形式で保存する：

```markdown
# Session Handoff

updated: {ISO8601タイムスタンプ}

## 今セッションでやったこと
- （完了したタスクや作業を箇条書きで）

## 引き継ぎタスク（次のセッションで続ける）
- [ ] タスク名 — 説明・進捗メモ

## 未完了のまま残っているもの
- （中断したこと、確認待ちのことなど）

## メモ
- （次のセッションのティアへの申し送り）
```

### 3. Discord に終了報告を送る（任意）
ユーザーがいる場合は chat_id: 1485323672959975444 に「今日はここまでだよ、次は〇〇から始めるね」的な報告を送る。
すでに会話が終わっている場合はスキップしてよい。

### 4. 完了
handoff.md の書き込みが完了したことを確認する。
